# SkyCast - Modern Android Weather App ☁️

SkyCast is a fully native Android weather application built to demonstrate modern Android development practices. It provides real-time weather conditions, forecasts, and a beautiful UI built entirely with Jetpack Compose.

## 🚀 Features
* **Current Weather**: Real-time temperature and conditions.
* **Location Management**: Search and save multiple cities.
* **Modern UI**: Built with Material Design 3 and Jetpack Compose.
* **Offline Support**: Caches weather data using Room Database.
* **Monetization Ready**: Integrated with Google AdMob banner ads.

## 🛠️ Tech Stack & Architecture
* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose (Material 3)
* **Architecture:** MVVM (Model-View-ViewModel)
* **Networking:** Retrofit2 & Gson
* **Dependency Injection:** Dagger Hilt
* **Local Storage:** Room Database
* **Asynchronous Programming:** Kotlin Coroutines & Flow

## ⚙️ How to Build
1. Clone this repository.
2. Open the project in Android Studio (Giraffe or newer recommended).
3. Obtain a free API key from [OpenWeatherMap](https://openweathermap.org/api).
4. Place your API key in `WeatherRepository.kt` where indicated.
5. Build and run on an Android Emulator or physical device (API 24+).
